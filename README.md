# Control Tower System

## 1. Overview
Hệ thống dashboard điều hành doanh nghiệp theo thời gian thực.

## 2. Business Goal
- Minh bạch dữ liệu
- Ra quyết định nhanh
- Kiểm soát KPI toàn hệ thống

## 3. Core Modules
- KPI Dashboard
- Alert System
- Drill-down Analytics
- Executive Reports
- SLA Monitoring
- Cross-system Integration

## 4. System Architecture
- Frontend: Filament Dashboard
- Backend: Laravel
- Data Source: ERP, MES, CRM
- Cache: Redis

## 5. Technology Stack
- Laravel 13
- Filament 5
- Livewire
- PostgreSQL

## 6. Data Model
- kpis
- metrics
- alerts
- logs
